Einführung in Cloud Computing

Was ist Cloud Computing?

Cloud Computing bezeichnet die Bereitstellung von IT-Ressourcen (wie Server, Speicher, Datenbanken, Netzwerke, Software) über das Internet („die Cloud“) auf Abruf.

Service-Modelle

  • IaaS (Infrastructure as a Service): Virtuelle Server, Netzwerke (z. B. AWS EC2, Azure VM)
  • PaaS (Platform as a Service): Entwicklungsplattformen (z. B. Heroku, Google App Engine)
  • SaaS (Software as a Service): Anwendungen über das Internet (z. B. Google Workspace, Microsoft 365)

Vorteile von Cloud Computing

  • Skalierbarkeit
  • Flexibilität
  • Kosteneffizienz (Pay-as-you-go)
  • Zugriff von überall
  • Automatische Updates und Wartung

Bekannte Anbieter

  • Microsoft Azure
  • Google Cloud Platform (GCP)
